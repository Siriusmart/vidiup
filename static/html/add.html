You can add an instance by making a request to https://vidiup.siri.sh/api/v1/add?region=[region]&instance=[instance].

  • Where [region] is the location of the instance (asia/europe/americas)
  • Where [instance] is the domain of the instance (e.g. invidious.fdn.fr)
